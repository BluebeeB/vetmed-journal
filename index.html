📁 JSON 파일로 복원

이전에 백업한 JSON 파일을 불러와 기존 데이터에 병합합니다. 중복 항목은 자동으로 제거됩니다.

✅ 권장 최종 구조 — 아이콘 한 번으로 앱 열기
GitHub Pages에 앱을 올리면:
📱 폰 홈화면 아이콘 → 앱 바로 실행 · AI 작동 · Drive 동기화 모두 가능
💻 PC 바탕화면 바로가기 → 브라우저에서 바로 열림
⚠️ API 키는 localStorage에 저장 — GitHub에 올라가는 HTML에 키가 없으므로 완전히 안전
🚀 GitHub Pages 배포 (10분, 한 번만)
1. github.com 가입 → 우측 상단 + → "New repository"
2. Repository name: vetmed-journal · Public 선택 → Create
3. "uploading an existing file" 클릭 → HTML 파일 드래그 → Commit changes
4. Settings → Pages → Branch: main → Save
5. 생성된 URL: https://[깃헙아이디].github.io/vetmed-journal/
6. 이 URL을 Apps Script 설정에도 사용 가능 → 동기화 완전 작동
⏱ 배포 후 1~2분 기다리면 URL이 활성화돼요
📱 홈화면 바로가기 추가
Android (Chrome): GitHub Pages URL 접속 → 주소창 옆 ⋮ → "홈 화면에 추가"
iPhone (Safari): GitHub Pages URL 접속 → 하단 □↑ 공유 → "홈 화면에 추가"
PC (Chrome): URL 접속 → 주소창 오른쪽 ⊕ 아이콘 → "설치" (또는 북마크바에 드래그)
✅ 이후엔 아이콘 탭 한 번으로 앱이 바로 열려요
🔄 앱 업데이트 방법
새 버전 HTML을 받았을 때:
① JSON 백업 다운로드 (기존 앱에서) → ② GitHub에 HTML 파일 재업로드 → ③ 새 앱에서 설정 탭에 API 키·Apps Script URL 재입력 → ④ 동기화 또는 JSON 복원
⚠️ Apps Script는 건드릴 필요 없어요.
Drive에 저장된 데이터와 Apps Script URL은 앱이 바뀌어도 그대로예요. URL만 기억해두면 돼요.

🔑 AI API 키 설정
왜 API 키가 필요한가요?
이 앱의 AI 기능은 Anthropic API를 직접 사용해요. Claude.ai 무료 플랜 한도와 완전히 별개예요.
API 키를 입력하면 앱을 로컬 파일로 저장해도 AI 기능이 정상 작동하고, 한도 걱정 없이 쓸 수 있어요.
⚠️ 키는 이 기기의 localStorage에만 저장되며, 외부 서버로 전송되지 않아요.
🔑 Claude API 키 발급 방법 (5분)
1. console.anthropic.com 접속 → 회원가입
2. "API Keys" → "Create Key" → 키 복사
3. 위 입력란에 붙여넣기 → 저장
💡 월 $1~3 예상 · 처음 가입 시 $5 크레딧 무료 지급
🆓 Google Gemini API 키 완전 무료
Gemini 2.0 Flash는 하루 1,500회, 분당 15회 완전 무료예요.
Claude API 키가 없거나 선택하지 않으면 Gemini를 사용해요. 수의학 학습 앱 용도로 충분한 성능이에요.
🔑 Gemini API 키 발급 방법 (3분)
1. aistudio.google.com/apikey 접속 (Google 계정)
2. "Create API key" 클릭 → 키 복사
3. 위 입력란에 붙여넣기 → 저장

📖 VetMed 일지 사용 가이드
신경내과학 · 소화기학 전공 대학원생을 위한 최적 사용 루틴을 정리했어요.
⚡ 매일 루틴 (5~10분)
①
강의·실습 중 단편 지식 퀵 입력 — Ctrl+Enter로 즉시 저장. 나중에 정리할 생각 말고 그때그때 1~2줄로.
②
귀갓길·자기 전 복습 알림 탭 확인 → 오늘 기록 중 복습 필요 항목 스스로 떠올려보기.
③
정보 출처 반드시 표시 — 교수님 말씀은 신뢰도 높음, 본인 추론은 "AI 정확도 검증"으로 확인.
🔬 주 2~3회 루틴 (20~30분)
①
단편 → 심화로 업그레이드 — 퀵 메모로 쌓인 단편 지식 중 중요한 것을 심화 학습으로 다시 작성. ✨ AI 구조 정리 활용.
②
AI 검색으로 연결 학습 — "CKD에서 고혈압 관리"처럼 오늘 배운 개념을 기존 기록과 연결해 검색.
③
퀴즈 5문제 — 틀린 문제는 자동으로 복습 목록에 추가됨. 오답 = 다음 심화 학습 주제.
📊 주말 루틴 (30~60분)
①
주간 AI 정리 — 주간 정리 탭에서 "✨ 주간 정리 생성". 이번 주 핵심 요약 + 다음 주 공부할 것 추천.
②
학습 전략 분석 — 학습 전략 탭 → "전체 학습 전략 분석". 월 1회 이상 실행해서 빈틈 영역 파악.
③
논문 추천 — 논문 추천 탭에서 신경내과/소화기 최신 논문 확인. MUST READ 논문은 저널클럽 준비에 활용.
④
JSON 백업 — 반드시 매주 1회. 백업 버튼 → JSON 다운로드 → Google Drive 저장.
🏥 증례 기록 활용법
실습·임상 실습 중 인상적인 증례를 기록할 때 사용해요. 환자 개인정보는 절대 입력 금지 — "12세 말티즈 중성화 수컷" 처럼 익명화.
"AI 교과서 연결" 기능으로 증례와 관련된 내 학습 기록을 자동 매칭해서 복습 포인트를 찾아줘요.
⚠️ 주의사항 · 보안
• 공용 PC에서 사용 금지 — 학습 기록이 localStorage에 그대로 남아요.
• AI 기능 = 인터넷 필요 — AI 버튼 사용 시 기록 일부가 Anthropic 서버로 전송돼요.
• 브라우저 캐시 삭제 주의 — 데이터 소실 가능. 최소 주 1회 JSON 백업 필수.
• API 키 보관 — 설정 탭의 API 키는 이 기기에만 저장. 다른 기기에서는 재입력 필요.
• AI 검증 결과도 오류 있을 수 있음 — 정확도 검증 결과는 교수님·교과서로 최종 확인.
